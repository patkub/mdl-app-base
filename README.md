Antuple Website
===============

Personal Website  
www.antuple.net

A Material Design Lite webapp backed by AngularJS.

### Download / Clone:

Clone the repo using Git:

```bash
git clone https://gitlab.com/patkub/antuple-website.git
```

### Local Setup:
```bash
npm install -g bower
npm install -g grunt-cli
npm install
bower update
grunt
```
